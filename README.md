# Lógica de Programação

Repositório contendo implementações de algoritmos de ordenação e busca em linguagem algorítmica.

## 📋 Arquivos

### Algoritmos de Ordenação

- **INSERTIONSORT.ALG** - Implementação do algoritmo Insertion Sort
  - Ordena um vetor de inteiros
  - Conta o número total de deslocamentos realizados
  - Complexidade: O(n²)

- **MERGESORT.ALG** - Implementação do algoritmo Merge Sort
  - Utiliza divisão e conquista
  - Implementa procedimentos para merge e recursão
  - Complexidade: O(n log n)

- **SelectioSort.ALG** - Implementação do algoritmo Selection Sort
  - Ordena um vetor de strings alfabeticamente
  - Busca o elemento mínimo em cada iteração
  - Complexidade: O(n²)

- **ORDENAMATRIZ5X5.ALG** - Ordenação de Matriz 5x5
  - Ordena linhas de uma matriz 5x5
  - Implementação do Insertion Sort para matrizes
  - Cada linha é ordenada independentemente

### Algoritmos de Busca

- **SISTEMA_BUSCA.ALG** - Sistema de Busca com Menu
  - Menu interativo com 4 opções
  - Implementa Bubble Sort para ordenação
  - Implementa Busca Binária para pesquisa
  - Status de ordenação do vetor

### Utilitários

- **POSICAOINSERIR.ALG** - Encontra Posição para Inserção
  - Determina a posição correta para inserir um valor
  - Mantém a ordem do vetor

## 🔧 Como Usar

Todos os arquivos estão em linguagem algorítmica (Pseudocódigo/Portugol) e podem ser executados em:
- VisuAlg
- Portugol Studio
- Outras IDEs que suportam pseudocódigo em português
